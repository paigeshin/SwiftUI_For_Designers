# Useful XCode Commands

### 1. Quick Insert from Library

command + shift + L

### 2. Toggle Lift Navigator

command + 0
command + 1

### 3. Show/hinde Preview

command + option + enter

### 4. Show/hide right panel

command + option + 0
command + option + 1

### 5. Fod and unfold code

command + option + <, >

### 6. Quick Action Menu

command + control + click

### Literals

`#imageLiteral()`

`#colorLiteral()`
